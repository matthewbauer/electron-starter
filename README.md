# GameTime
GameTime is like Popcorn Time for retro games.

## Run

Download the latest packaged version from the releases page.

## Build
`git clone https://github.com/matthewbauer/gametime.git`

`cd gametime`

`npm install`

`./script/build`

`./script/run`
